# Claude Sandbox

A repository for exploration and experimentation. When you have an idea you want to try out or a concept you want to explore, this is the place.

## Structure

Each exploration lives in its own directory with its own README explaining:
- What the exploration is about
- What we learned
- Any relevant notes or future directions

```
/explorations
  /exploration-name
    README.md          # Purpose and learnings
    ...                # Implementation files
```

## Philosophy

- **Learn by doing**: The goal is exploration, not perfection
- **Document as you go**: Each exploration should be self-explanatory
- **Keep it simple**: Start minimal, add complexity only when needed
- **Iterate freely**: It's okay to try things that don't work out

## Getting Started

To start a new exploration, create a new directory under `/explorations` with a descriptive name and add a README explaining what you're exploring.
