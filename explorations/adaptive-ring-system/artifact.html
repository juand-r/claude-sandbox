Adaptive Ring System

256 self-modifying rings. Each ring is 36 bits and is at once data, an elementary-CA rule, and a node in a transformation network. No controller, no fitness, no energy — births, deaths, mutation, and a hard cap only.

SLOTS — colour = RULE (position is address, not space)

Each cell is one of the 256 slots. Colour = that ring's 8-bit RULE. Dark = empty. Adjacent cells are not neighbours; rings interact by address via pull/push.

GENOMES — one row per slot, 36 bits

METRICS

green = population, blue = unique genotypes (of 256).
