# Time Perception — Overnight Report

Built and tested a self-contained model of subjective time out of interacting
oscillations (no LLM). This report covers what was built, what the experiments
showed, the things that surprised me (and forced corrections), and where I think
this should go next. Figures referenced are regenerated by the scripts named in
each section (they are git-ignored; I send the PNGs alongside this report).

TL;DR of the science:
1. The dissociation works and is robust: the same oscillator system makes idle
   time *drag* in the moment yet busy time be remembered as *longer*.
2. The scalar property (Weber) requires **common-mode** (global) rate noise.
   Independent per-oscillator noise does *not* give it — it degrades the code.
3. Behavioural psychophysics (geometric-mean bisection, Vierordt central
   tendency) live in a **log/ratio read-out**, not in the oscillator kernel.
   Oscillations are necessary but not sufficient.
4. Coupling earns its place only in a **narrow weak-coupling window**; past the
   synchronization transition it collapses the multi-scale code entirely.
5. Two multi-scale codes, opposite failure modes: the **cascade** (nested loops)
   is compact and exact but catastrophically brittle; the **incommensurate bank**
   is robust but bounded in range. Same lesson twice: capacity trades off against
   robustness.
6. That trade-off is resolvable: **local (adjacent) coupling stabilises the
   brittle cascade** (~90x lower error under drift) without the code collapse that
   *global* coupling causes. Locality is the key distinction.

## 1. Architecture

One engine, several read-outs. The substrate is a bank of phase oscillators at
geometric frequencies. Events perturb the phases (perception). An attention gate
modulates a prospective clock. Read-outs tap the same state.

```
events --> [attention gate] --> pulses
                                  |
        +----------+----------+---+--------------+
        |          |          |                  |
   prospective  retrospective  coherence     phase code
   tau          load          (recency)     (multi-scale)
   (felt rate)  (memory)
        |          |                              |
   "idle drags / busy flies"            duration estimate,
                                        bisection, central tendency
                                        (via a LOG observer on top)
```

Files:
- `oscillators.py`   — Stage 1 engine (`TimePerceptionSystem`, `Params`).
- `readouts.py`      — SBF coincidence substrate + log observer.
- `coupling.py`      — Kuramoto-coupled bank (Stage 2).
- `cascade.py`       — hierarchical odometer cascade (nested loops).
- `experiments*.py`, `robustness.py` — the experiments below.

## 2. Stage 1 — the dissociation (figure: `stage1_results.png`)

The headline phenomenon. Over the same 60 s window, a **busy** stream (many
events) and an **idle** stream (few events):

| read-out                  | busy   | idle   | interpretation                |
|---------------------------|--------|--------|-------------------------------|
| prospective felt time     | ~6 s   | ~43 s  | idle **drags**, busy **flies** |
| retrospective load        | ~4300  | ~340   | busy remembered **longer**     |

The two read-outs disagree by construction of opposite gates: the prospective
clock runs *fast when little is happening* (attention is free to watch the clock);
retrospective load simply accumulates events. This is the holiday paradox, and it
is the reason the system is more than a timestamp counter.

Idle detection (self-regulation): a "check-in" fires when *felt* idle time crosses
a threshold — using subjective, not wall-clock, idle time.

### Robustness (figure: `robustness_results.png`)
Not a knife-edge. Sweeping event rate 0.05 -> 8 /s over the window, felt time
decreases monotonically (54.7 -> 2.8 s) while load increases monotonically
(91 -> 7034). "Busy flies" (negative felt-vs-rate slope) holds across every tested
`gate_k` (0.5-3) and `activity_tau` (0.5-4); those only change the steepness.

---

### Self-regulation loop closed (figure: `selfreg_results.png`)
Feeding check-ins back as self-events makes the loop dynamic. With no external
input the agent self-paces at a **regular limit cycle** (interval 9.82 s, std
0.003 s for a felt threshold of 8 s). The period exceeds the threshold because a
check-in bumps activity and lowers the gate, so felt time accrues slowly until the
activity leaks away — the period is *emergent* from the gate dynamics, not just the
threshold. Background activity **suppresses** the rhythm smoothly (check-in rate
0.10/s when idle -> ~0 once background exceeds ~0.8/s): the agent checks in less
when much is happening. This is context-gated self-pacing.

## 3. Stage 1b — psychophysics, and where it actually lives
   (figure: `stage1b_results.png`)

I tried to get the three classic signatures out of the oscillator coincidence
read-out (SBF). Two of three did **not** come from the oscillators, and finding
out why was the most useful part of the night.

### Scalar property (Weber's law)
- **Common-mode** frequency jitter (one global rate fluctuation per trial) -> CV
  flat at ~0.05 across 2-32 s. Scalar. This is exactly the SET claim: trial-to-trial
  variation in pacemaker rate.
- **Independent** per-oscillator jitter -> CV *grows* 0.03 -> 0.23. It scrambles
  the phase pattern (phase error ~ f·eta·T exceeds pi at long T) instead of
  rescaling time. So the *kind* of noise is the whole story; only global
  multiplicative noise gives Weber.

### Bisection
- The SBF coincidence rule **fails** to bisect: with only two anchor templates
  (4 s, 16 s), a probe in between is farther than ~1/f_max from *both*, so both
  similarities collapse to ~0 and the decision is noise-driven (point lands at a
  meaningless ~4.7 s). The native resolution is **absolute** (set by f_max), not
  scalar — so the kernel has no notion of ratio.
- The **log observer** (decide "long" if the log-measurement exceeds the
  log-midpoint) lands the 50% point on the **geometric mean** (7.98 vs 8.0).

### Central tendency (Vierordt)
- The log observer with a Gaussian prior over the range gives regression to the
  mean: log-log slope **0.79** (<1), indifference point 8.5 ≈ geometric mean of
  the range. Short overestimated, long underestimated.

**Conclusion:** the oscillator substrate provides the multi-scale code and (with
common-mode noise) scalar variability, but the ratio-based psychophysics
(geometric-mean bisection, Vierordt) require a logarithmic read-out on top. This
vindicates the log-scale recommendation from the initial design discussion: the
clean place to put scalar/ratio behaviour is a log time axis.

## 4. Stage 2 — does coupling earn its place?
   (figure: `stage2_results.png`)

Kuramoto mean-field coupling, 24 oscillators, f in [0.5, 4] Hz.

- **Synchronization transition** at K_c ≈ 8 (steady-state order parameter jumps
  0.2 -> ~0.9).
- **Coding horizon** (first recurrence of the start pattern = how long the code
  stays unique) **collapses at the same K**: >30 s (censored) for K<=7.5, then
  25.7 (K=8) -> 3.4 (K=9) -> 0.46 (K>=12). Strong coupling locks everything to one
  frequency and the multi-scale code dies.
- **Recency** (order parameter 2 s after a reset): 0.10 uncoupled (decays fast);
  rises with K. Sweet spot **K ≈ 6-8**: recency sustained ~3x longer while the
  horizon is still intact (>=25 s). Above K ≈ 9 the code is gone.

**Verdict:** coupling buys a longer-lasting recency signal, but only in a narrow
weak-coupling window just below K_c. Above it, it is catastrophic. This confirms
the Stage-1 skeptical prediction exactly.

## 5. Nested loops — the cascade (figure: `cascade_results.png`)

The cascade is your "nested feedback loops at different scales" made literal: a
ladder of oscillators each M× slower = an odometer / mixed-radix code for time.

- **Capacity:** 4 stages (M=10) decode a 1000 s range *exactly* (noiseless max
  error 0). Exponential range from a linear number of oscillators. A single
  oscillator just aliases.
- **Brittleness:** catastrophically fragile. Per-oscillator phase noise of 0.01
  already gives off-by-1 s errors; 0.02 -> off-by-10 s. A flipped digit costs M^k.
  90th-percentile error: 0 -> 1 -> 10 -> 19 s as noise rises.
- **Incommensurate bank, same range:** degrades **gracefully** — 90th-percentile
  error stays 0.02 -> 0.06 s across the same noise. Robust, but needs more
  oscillators and has a bounded unique range.

Two opposite strategies for multi-scale time: cascade = compact + exact + brittle
(digital/positional); bank = redundant + graceful + bounded (analog). This is the
same capacity-vs-robustness trade-off as the coupling result, in another guise.

## 5b. Structured coupling — the synthesis (figure: `structured_results.png`)

I pursued the open lead directly: F3 said global coupling destroys the code, F4
said the cascade is brittle. The natural resolution is **local** coupling — lock
only adjacent scales, M:1, so each slow stage is slaved to a sub-harmonic of its
faster, more accurate neighbour.

With 3% per-stage frequency drift over a 60 s run:
- **unlocked** cascade: 90th-pct decode error **7.25 s** (digit flips).
- **adjacent locking (K=2):** **0.08 s** — a ~90x improvement.
- **K=32:** still **0.08 s** — local coupling does **not** collapse the code even
  when strong, unlike the global coupling of Section 4 (which collapses past K_c≈8).
- Drift tolerance: unlocked error grows to ~18 s at 12% drift; locked stays <0.3 s.

This resolves the capacity-vs-robustness tension: the brittle, high-capacity
cascade becomes robust when stabilised by *structured* (local) coupling. The
distinction that matters is locality — adjacent coupling preserves the frequency
ladder; all-to-all coupling erases it. (Residual: occasional ~4 s glitches at
digit boundaries remain even when locked.)

## 6. Honest limitations

- **Scalar property is partly "by construction."** In both the Stage-1 accumulator
  and the SBF read-out, Weber behaviour is *injected* as common-mode multiplicative
  noise. That is the correct and standard assumption, but it is an assumption, not
  an emergent property of the dynamics.
- **The behavioural read-out is not oscillatory.** Geometric-mean bisection and
  Vierordt come from a log/Bayesian observer, not from oscillations. The "everything
  is oscillations" aesthetic hits a real wall here.
- **Retrospective load is a crude proxy** (summed perturbation magnitude). It
  reproduces the sign of the holiday paradox but is not calibrated to anything.
- **Arousal/attention remains a knob**, as flagged from the start. It shifts
  estimates the right way but tests parameterization, not explanatory content.
- **Coupling experiments use a moderate band** so the transition is reachable; a
  very wide (truly multi-scale) band would need much larger K to sync. The
  qualitative collapse-at-K_c result is band-independent, the numbers are not.
