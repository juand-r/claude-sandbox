## Abstract / Summary

Presents an **acoustically emotion-aware conversational agent** that enables speech emotion recognition and stylizes responses with empathetic feedback and interjections. Most prior CA work uses NLP/text-based techniques for affective interaction; this work focuses on **acoustic features of speech** for voice-based CAs.

Experiment with 75 participants evaluated perceived emotional intelligence (PEI) after interacting with the CA. Results:
- Acoustical emotion-awareness **increased participants' PEI** of the CA
- Empathetic responses helped **alleviate some participants' negative emotions**

## Key Concepts

- **Acoustic emotion recognition**: Using speech prosody, tone, rhythm to detect user emotional state (not just text content)
- **Empathetic response generation**: CA generates responses styled with empathetic feedback and interjections
- **Perceived emotional intelligence**: Measure of how emotionally intelligent users perceive the agent to be
- **Multimodal emotion sensing**: Goes beyond text to acoustic channel
- **Emotion alleviation**: The agent's empathetic responses have measurable effect on user emotional state

## Relevance to Cybernetics-Agents Research

Relevant for understanding the cybernetic loop in human-agent interaction:
- **Sensing-acting loop**: Agent senses user emotion (acoustic input) → generates empathetic response (output) → affects user emotion (feedback) — this is a classic control loop
- **Social regulation**: The agent is actively regulating the user's emotional state, not just responding to it
- **Multimodal interoception analog**: Acoustic emotion recognition is the agent's way of "reading" the user's internal state — analogous to interoception but in the social domain
- **Measurable effect**: The fact that empathetic responses actually alleviate negative emotions confirms the regulatory function
