# A Computational Constructivist Model as an Anticipatory Learning Mechanism for Coupled Agent-Environment Systems

**Author:** Filipo Studzinski Perotto
**Year:** 2013
**Journal:** Constructivist Foundations, Vol. 9, No. 1, pp. 46-56
**URL:** https://constructivist.info/9/1/046.perotto
**Type:** Target article in special issue on Computational Constructivism

## Core Contribution

Introduces CALM (Constructivist Anticipatory Learning Mechanism) — an agent learning mechanism based on constructivist AI principles. This is one of the few actual computational implementations of constructivist epistemology.

## Key Arguments

- CALM models regularities in agent-environment interaction at both the sensorimotor level and through construction of abstract/high-level representational concepts.
- Introduces the CAES (Coupled Agent-Environment System) meta-architecture: defines an autonomous agent that is situated, embodied, and intrinsically motivated.
- Uses factored partially observable Markov decision processes (FPOMDP) for model-based learning.
- The agent constructs its knowledge through interaction rather than receiving pre-defined representations of the world.
- Offers an alternative to behaviorist and probabilistic approaches dominating AI.

## Architecture Details

- Agent does not treat input as "perception of a pre-given world" (anti-realist)
- Learning through regularities in stream of experience
- Intrinsic motivation drives exploration rather than programmer-defined extrinsic goals
- Anticipatory: agent builds models that predict future states

## Relevance to Project

This is a concrete computational artifact built on constructivist principles. The CAES architecture (situated, embodied, intrinsically motivated) maps directly onto the kind of agent design we're interested in. The tension between using formal mathematical tools (FPOMDP) while maintaining constructivist epistemology is instructive.
