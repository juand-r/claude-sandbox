## Abstract / Summary

Proposes a unified theory and model of human behavior integrating **physiology and primal affect** with cognitive processes in a cognitive architecture. Extends ACT-R (Adaptive Control of Thought — Rational) with physiological and affective subsystems to create ACT-RΦ.

Affect is treated as a **lower-level, functional process** that interacts with cognitive processes (especially declarative memory) to produce emotional behavior. The architecture provides specific, quantitative predictions about affect-cognition interactions.

## Key Concepts

- **ACT-RΦ architecture**: Extension of ACT-R with physio, SEEKING, and Affective-Associations modules
- **HumMod integration**: Connects HumMod physiological simulation system to ACT-R
- **Primal affect based on Panksepp**: SEEKING system from Panksepp and Biven (2012) — basic emotional circuits
- **Physiology → Affect → Cognition pipeline**: Physiological states influence affect, which modulates cognition
- **Functional layer**: Affect serves as intermediary between physiological substrate and cognitive processing
- **Backward compatible**: Uses ACT-R's existing module system

## Relevance to Cybernetics-Agents Research

This is perhaps the most directly cybernetic paper in this collection:
- **Physiology as the ground layer**: The agent has a simulated body (HumMod) with homeostatic processes
- **Affect as a regulatory signal**: Affect is explicitly positioned as a functional layer between body-regulation and cognition — this IS the cybernetic loop
- **Primal affect (Panksepp)**: SEEKING, RAGE, FEAR, etc. are basic regulatory circuits, not high-level cognition
- **Quantitative predictions**: The architecture makes testable predictions, not just qualitative analogies
- Directly connects to interoception (body state → affect) and allostasis (predictive regulation)

## Architecture Layers
1. Physiological substrate (HumMod) — homeostatic processes, drives
2. Primal affect system (SEEKING, basic emotions) — regulatory signals
3. Cognitive system (ACT-R) — declarative memory, production rules
4. Behavior — output of the integrated system
