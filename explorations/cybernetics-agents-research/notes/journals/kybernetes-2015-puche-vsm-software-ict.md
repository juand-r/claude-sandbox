# Puche Regaliza (2015) - Quantitative Analysis of VSM on Software Projects in ICT

**Citation:** Puche Regaliza, J.C. (2015), "Quantitative analysis of viable systems model on software projects in the ICT sector in Castilla y Leon", *Kybernetes*, Vol. 44 No. 5, pp. 806-822.

**DOI:** 10.1108/K-06-2014-0112

**Affiliation:** Universidad de Valladolid, Spain

**Access:** Emerald Insight (paywalled). No preprint found.

## Purpose

Detect the degree of influence between applying VSM on a software project and its viability/success.

## Method

- Quantitative: identified theoretical constructs for VSM Systems 1-5 and software project viability indicators
- Developed a standardized questionnaire used for data collection from ICT companies in Castilla y Leon
- Built a structural equation model (SEM) from the collected data

## Key Findings

- The SEM results show measurable relationships between VSM system functions and software project success
- Provides empirical (not just theoretical) evidence that VSM principles correlate with viable software projects
- One of the few papers that directly applies VSM to software project management with quantitative validation

## Relevance to Our Project

This is one of the most directly relevant papers: it operationalizes VSM constructs into measurable indicators for software projects. If we want to design agent systems using VSM principles, we need to know which VSM functions matter most for software viability. Puche Regaliza's work gives us a starting point for that mapping.

The structural equation model approach could inform how we validate whether cybernetic design principles actually improve agent system outcomes.
