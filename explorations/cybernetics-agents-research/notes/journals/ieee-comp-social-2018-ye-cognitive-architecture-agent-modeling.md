## Summary

Proposes a general cognitive architecture for agent decision-making in artificial societies (multi-agent simulations of complex social systems). The core problem: different application scenarios focus on different facets of decision-making, which hinders model reuse and system integration. The architecture attempts to unify these facets into a single framework.

## Key Contributions

1. **Unified cognitive architecture**: Covers perception, memory, learning, decision-making, and action in a single agent framework. Designed so that different simulation programs can be reorganized and integrated conveniently.

2. **Artificial societies as analytical foundation**: Positions artificial societies (implemented via multi-agent approaches) as the foundational method for analyzing complex eco- and social systems.

3. **Demonstration via two simulations**: Emergent evacuation behavior and population evolution, showing the architecture's flexibility across domains.

## Relevance to Cybernetics Research

- Directly addresses the agent architecture question central to computational social systems
- The "artificial societies" concept (from Fei-Yue Wang's ACP framework) is a cybernetic approach: build parallel artificial systems, run computational experiments, compare with reality
- The cognitive architecture itself embodies feedback loops: perception -> cognition -> action -> environment change -> new perception
- Connects to second-order cybernetics: observers (agents) constructing models of their world and acting on those models
