# Information Decomposition Based on Cooperative Game Theory

**Authors:** Nihat Ay, Daniel Polani, Nathaniel Virgo
**Journal:** Kybernetika, Vol. 56, No. 5, pp. 979-1014, 2020
**DOI:** 10.14736/kyb-2020-5-0979
**URL:** https://www.kybernetika.cz/content/2020/5/979
**PDF:** https://www.kybernetika.cz/content/2020/5/979/paper.pdf
**Keywords:** information geometry, partial information decomposition, cooperative game theory

## Summary

Proposes "Information Attribution" (IA), a new method for decomposing mutual information
using cooperative game theory (Shapley values). Bridges information theory and game theory
by treating information contribution analogously to player payoff contributions.

## Core Framework

- Variables are treated as "players" in a cooperative game
- Shapley values fairly allocate mutual information across source variable combinations
- Uses an alternative lattice structure compared to standard PID (partial information decomposition)
- Fewer decomposition terms: preserves synergy and unique information, but integrates
  redundant information into unique information terms

## Key Properties

- Satisfies "local positivity" and "identity" axioms simultaneously
- These cannot be satisfied together by traditional PID measures
- Provides mathematically rigorous decomposition respecting fairness and efficiency axioms

## Information Decomposition Components

- **Unique information:** what only one variable provides
- **Redundant information:** what multiple variables provide identically (folded into unique info here)
- **Synergistic information:** what requires multiple variables jointly

## Relevance to Cybernetics-Agents Research

This is a deeply cybernetic paper -- it formalizes how information flows in multi-source systems:
- **Information-theoretic control:** understanding what information each agent/sensor contributes
- **Multi-agent systems:** the game-theoretic framing maps directly to multi-agent settings
- **Modern cybernetic formalism:** updates Ashby-style information-regulation ideas with
  rigorous game-theoretic and information-geometric tools
- **Regulation insight:** if you want to know which sensors/channels matter for control,
  this decomposition tells you what each contributes uniquely vs synergistically

The authors (Ay, Polani, Virgo) are all deeply connected to the cybernetics/complex systems
community. Polani in particular has extensive work on information-theoretic approaches to
embodied agents and empowerment.
