# Seal Mystery Game - Development Plan

## Concept
Sarah discovers a mysterious seal in the Bolivian desert, leading to uncovering a hidden civilization.

## Technical Approach
- Browser-based game (HTML5 Canvas + JavaScript)
- Single HTML file with embedded CSS and JS for simplicity
- No external dependencies required

## Game Structure

### Scenes
1. **Desert Arrival** - Sarah arrives at the dig site, atmospheric introduction
2. **The Discovery** - Finding the seal in its air bubble
3. **The Awakening** - Seal comes alive, shadowy figure appears
4. **The Descent** - Entry into underground civilization
5. **The Puzzle Chamber** - Ancient puzzles to solve
6. **The Truth** - Final reveal about the civilization's purpose

### Puzzles
- [x] Symbol matching puzzle (ancient glyphs)
- [x] Light reflection puzzle (mirrors and beams)
- [x] Sequence memory puzzle (seal movements)

### Art Style
- Minimalist, silhouette-based visuals
- Limited color palette: deep blues, golds, sandy browns
- Heavy use of shadows and atmospheric effects

### Audio
- Web Audio API for ambient sounds
- Procedural eerie tones

### Core Mechanics
- Point-and-click exploration
- Dialogue system
- Inventory (minimal)
- Puzzle interaction

## The Final Reveal (Spoiler)
The seal is actually a guardian, and the civilization seeded human consciousness millions of years ago. The shadowy figure is Sarah's future self, ensuring the timeline remains intact. The "seal" is not an animal but a literal seal - a lock on a doorway between dimensions.
