# The Seal of Eternity

A narrative puzzle game about Sarah, an archaeologist who discovers a mysterious seal in the Bolivian desert.

## How to Run

Open `index.html` in a modern web browser. No server required.

## Controls

- Click to interact with glowing objects
- Click dialog box or press Space/Enter to advance text
- Press R at the end to restart

## Story

Sarah discovers an impossible artifact in the Bolivian Altiplano: a seal (the animal) preserved in a bubble of air for centuries. When she touches it, the seal awakens, and a shadowy figure emerges. Together they lead her underground to an ancient civilization that has been watching humanity since its inception. The final reveal will change everything she thought she knew.

## Features

- 5 narrative scenes with full dialog
- 3 puzzle types: Symbol matching, light reflection, memory sequence
- Procedural ambient audio using Web Audio API
- Particle effects and atmospheric visuals
- Silhouette-based art style with limited color palette

## Technical

- Pure HTML5/CSS3/JavaScript
- No external dependencies
- Canvas-based rendering
- Responsive design
