# Prompt Generator

Generates random, surprising project prompts for future explorations by combining elements from different categories.

## Run

```bash
python generator.py           # Generate one prompt
python generator.py -n 5      # Generate multiple prompts
python generator.py --seed 42 # Reproducible output
```

## How It Works

Combines random elements from:
- **Domains** - What area/field
- **Formats** - What form it takes
- **Constraints** - Interesting limitations
- **Twists** - Unexpected angles

Edit the word lists in `generator.py` to customize.
