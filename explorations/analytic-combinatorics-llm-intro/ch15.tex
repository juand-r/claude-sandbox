\chapter{Phase Transitions in Language Model Output}
\label{ch15}

This chapter asks whether temperature variation in language models can reveal genuine large-scale structural changes in the output distribution. The answer has two layers.

\begin{enumerate}
  \item There is a clean mathematical object: the \emph{global} partition function
  \[
    Z(\beta) = \sum_{w \in \Sigma^*} \mu(w)^\beta.
  \]
  \item There is a practical experimental procedure: vary the \emph{local} decoding temperature used by an LLM and measure changes in its outputs.
\end{enumerate}

Chapter~14 showed that those are not the same thing in general. So the purpose of this chapter is not to pretend the bridge is already proved. It is to explain the global object, summarize what singularity ideas would matter if the right surrogate were available, and then interpret the empirical evidence as suggestive rather than decisive.

\section{The Global Partition Function}

Let $\mu$ be the output distribution of a tight autoregressive model on content strings $w \in \Sigma^*$. For inverse temperature $\beta>0$, define
\[
  Z(\beta) = \sum_{w \in \Sigma^*} \mu(w)^\beta.
\]
This is a Renyi-type partition sum over complete strings.

Basic facts are immediate.
\begin{itemize}
  \item If the model is tight, then
  \[
    Z(1) = \sum_w \mu(w) = 1.
  \]
  \item For every $\beta \ge 1$,
  \[
    Z(\beta) \le 1,
  \]
  because each term satisfies $0 \le \mu(w)^\beta \le \mu(w)$.
  \item For $0<\beta<1$, finiteness is not automatic. Small probabilities are enlarged by the exponent, so the sum can in principle diverge.
\end{itemize}

The corresponding global Gibbs distribution is
\[
  \mu_\beta(w) = \frac{\mu(w)^\beta}{Z(\beta)}
\]
whenever $Z(\beta)$ is finite.

\section{Why This Is Not an Ordinary OGF}

A major conceptual caution is needed here. The function $Z(\beta)$ is not automatically an ordinary generating function of the Chapter~4 form
\[
  \sum_{n \ge 0} a_n z^n
\]
with integer exponents. If we write
\[
  E(w) = -\log \mu(w),
\]
then formally
\[
  Z(\beta) = \sum_w e^{-\beta E(w)}.
\]
Setting $z=e^{-\beta}$ gives an expression of the form
\[
  \sum_w z^{E(w)},
\]
where the exponents are generally arbitrary nonnegative real numbers, not integers. So Chapter~4's transfer theorems do \emph{not} apply directly just by the substitution $z=e^{-\beta}$.

What survives from earlier chapters is more limited but still useful:
\begin{itemize}
  \item if one can replace the true LLM partition sum by a finite-state or algebraic \emph{surrogate}, then the singularity toolkit of Part~I becomes relevant to the surrogate;
  \item if not, then one should regard the Chapter~4 language as analogy, not theorem.
\end{itemize}

\section{What We Mean by a Phase Transition Here}

For a mathematically defined global Gibbs family, the cleanest notion is:
\begin{quote}
A phase transition is a point where the partition function or a thermodynamic quantity built from it ceases to behave analytically as a function of $\beta$.
\end{quote}

In finite-size empirical work, one never sees a literal nonanalyticity. Instead one sees sharp crossovers, peak susceptibilities, and finite-size scaling patterns that are interpreted as evidence for an underlying infinite-limit singularity.

For this reason we will be cautious with first-order / second-order terminology. The relevant derivative tests depend on which free-energy normalization one chooses and on what limiting family of systems is being studied. This chapter will therefore emphasize \emph{qualitative singularity types} and empirical signatures rather than trying to force a fragile derivative classification onto every example.

\section{A Prototype Algebraic Critical Point}

The Catalan family remains the cleanest model example. Its counting generating function is
\[
  C(z) = \frac{1-\sqrt{1-4z}}{2z},
  \qquad
  [z^n]C(z) \sim \frac{4^n}{\sqrt{\pi}n^{3/2}}.
\]
If we build the size-weighted partition sum
\[
  Z_C(\beta) = \sum_{n \ge 0} C_n e^{-\beta n},
\]
then the critical inverse temperature is
\[
  \beta_c = \log 4.
\]
At criticality,
\[
  C_n e^{-\beta_c n} \sim \frac{1}{\sqrt{\pi}n^{3/2}}.
\]
So the critical Boltzmann weights are heavy-tailed with the characteristic exponent $-3/2$. This is a clean combinatorial model of algebraic criticality.

Two cautions are essential.
\begin{enumerate}
  \item For $\beta < \beta_c$, the physical sum $Z_C(\beta)$ diverges. Analytic continuation of the function beyond that boundary is a different mathematical object, not the original partition sum.
  \item This example is about a counting-by-size partition sum, not directly about the LLM partition sum $\sum_w \mu(w)^\beta$.
\end{enumerate}

So Catalan provides a \emph{prototype} of square-root criticality, not a proof that LLM temperature decoding sits in the same universality class.

\section{Composition Schemes as a Catalogue of Possibilities}

Results such as those of Banderier, Kuba, Wagner, and Wallner \cite{BanderierKubaWallner2021} show that in combinatorial models, critical composition schemes can produce a small catalogue of recurring singular behaviors: poles, square-root branch points, and more degenerate Airy-type phenomena.

For this book, the right takeaway is not that those theorems already apply to LLM partition sums. It is that they supply a menu of plausible universality patterns one might later try to fit or motivate if an appropriate surrogate model is found.

\section{What the Empirical Papers Actually Vary}

The recent empirical papers on LLM ``phase transitions'' do \emph{not} directly sample from the global Gibbs distribution
\[
  \mu_\beta(w) \propto \mu(w)^\beta.
\]
They vary the practical decoding temperature used locally at each token. Chapter~14 showed that this usually produces a different distribution.

So the empirical story should be read carefully:
\begin{itemize}
  \item the experiments are about \emph{local temperature decoding},
  \item the global partition function provides a conceptual reference object,
  \item and the relation between the two is part of the open theory, not a settled equivalence.
\end{itemize}

This does not make the experiments irrelevant. It means they are evidence about a nearby but not identical object.

\section{Empirical Evidence from LLMs}

Several recent studies report sharp temperature-dependent changes in output statistics for models such as GPT-2, Pythia, Mistral, and Llama.

\paragraph{Nakaishi, Nishikawa, and Hukushima (2024).}
They study GPT-2 across a range of local decoding temperatures and observe a sharp crossover in susceptibility-like observables near a critical window around $T \approx 1$ \cite{NakaishiNishikawaHukushima2024}.

\paragraph{Arnold et al. (2024).}
They extend the picture across model families and scales, again finding robust temperature windows and critical-looking scaling behavior \cite{ArnoldEtAl2024}.

\paragraph{Mikhaylovskiy (2025).}
This work goes further and proposes a specific universality-class interpretation for those fitted exponents \cite{Mikhaylovskiy2025}. That proposal is interesting, but still empirical and model-dependent.

The honest summary is this: the papers present suggestive finite-size evidence for phase-transition-like behavior under local temperature decoding. They do not yet prove that the global partition function $Z(\beta)$ has a specific singularity.

\section{The Bridging Proposal}

Here is the research proposal that makes the chapter worth having.

\begin{quote}
Perhaps the sharp empirical crossovers seen under local temperature decoding are shadows of a deeper global singularity structure, which in turn might be captured by a rational or algebraic surrogate for the LLM output distribution.
\end{quote}

That proposal has several possible forms.
\begin{itemize}
  \item In a rational surrogate, the dominant analytic structure would be pole-like.
  \item In an algebraic surrogate, square-root or related branch-point behavior could appear.
  \item More complicated surrogates could produce richer singularity landscapes.
\end{itemize}

What the proposal does \emph{not} say is that one may apply Chomsky--Sch\"utzenberger directly to
\[
  Z(\beta)=\sum_w \mu(w)^\beta.
\]
The counting theorems of Chapters~5--6 concern different objects. Any bridge from PCFG-style algebraicity to LLM partition sums would require a new weighted theorem or a new surrogate argument, not a direct reuse of the old counting theorem.

So the bridging proposal should be read as conjectural. It is a disciplined conjecture, but still a conjecture.

\section*{Summary}

This chapter's safe conclusions are:
\begin{enumerate}
  \item the global partition function of an LLM is
  \[
    Z(\beta)=\sum_w \mu(w)^\beta;
  \]
  \item this object is not automatically an ordinary generating function in an integer index;
  \item local decoding temperature and global Gibbs temperature are different objects;
  \item empirical temperature studies therefore provide suggestive but indirect evidence about the global partition function;
  \item combinatorial models such as Catalan structures show what algebraic criticality could look like if the right surrogate were found.
\end{enumerate}

That is enough to support a serious theory-meets-evidence program without pretending the bridge is already finished.

\section{Exercises}

\begin{exercise}
Show directly that if $\mu$ is a probability distribution on a countable set, then for every $\beta \ge 1$,
\[
  \sum_w \mu(w)^\beta \le 1.
\]
Why does the same argument fail for $0<\beta<1$?
\end{exercise}

\begin{exercise}
For the Catalan partition sum
\[
  Z_C(\beta)=\sum_{n \ge 0} C_n e^{-\beta n},
\]
identify the critical inverse temperature and show that the critical weights behave like $n^{-3/2}$.
\end{exercise}
