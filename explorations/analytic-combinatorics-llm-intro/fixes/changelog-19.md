# Chapter 19 Changelog

## What I changed

- Repaired the roadmap chapter so it now distinguishes much more clearly between classical exact toolkit, surrogate-based verification programs, and genuinely conjectural LLM bridges.
- Softened the “where we stand” summary of Part IV so it no longer sounds as if the global partition-function / phase-transition bridge is already theorem-complete.
- Made Gap 9 the explicit starting point for a **chosen approximation target**, not just “some WFA-like thing,” and propagated that object-discipline into the three-phase program.
- Rewrote the Gap 1b roadmap discussion so it no longer treats the entropy/radius identity as an immediate corollary once a WFA exists. It is now framed as a surrogate-verification problem that depends on choosing the right counting object.
- Added explicit warnings about the local-vs-global temperature distinction in the prize-problem and Phase 3 discussions, so the roadmap no longer silently identifies empirical decoding temperature with the global Gibbs parameter.
- Strengthened the caution around Gap 4 so an empirical `n^{-3/2}` fit is treated as suggestive rather than as direct proof of approximate context-freeness.
- Added a student-facing note that some earlier chapters are classical exact toolkit while others are bridge chapters still containing open research components.

## Note items addressed

- Addressed the main consistency problems between the old roadmap and the repaired Chapters 9, 12, 14, and 15: entropy/counting-object drift, approximation-target ambiguity, and local/global temperature confusion.
- Preserved the useful tractability ranking and three-phase structure, while making the claims about what each phase would actually prove more accurate.

## Pushback and deferrals
