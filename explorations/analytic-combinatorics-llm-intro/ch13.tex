\chapter{Boltzmann Sampling}
\label{ch13}

This chapter has one beautiful exact insight at its center:
\begin{quote}
For a combinatorial class counted by size, Boltzmann sampling is exactly a Gibbs distribution with energy equal to size, and the ordinary generating function is exactly the partition function.
\end{quote}
That statement is precise, not metaphorical. Everything else in the chapter should be read as built on top of that fact.

\section{Motivation}

Suppose we want a uniform random object of size exactly $n$ from a combinatorial class $\mathcal{A}$. A classical approach uses a recursive sampler together with precomputed counting tables. Boltzmann sampling takes a different route: instead of forcing the size to be exactly $n$, it samples from all sizes at once with a tunable parameter $x$ and then, if necessary, conditions or rejects to recover exact size.

The main point for this book is not the last word on complexity bounds. It is the formula for the distribution itself and how naturally it interacts with generating functions.

\section{Boltzmann Samplers}

Let
\[
  A(x)=\sum_{n \ge 0} a_n x^n,
  \qquad a_n = |\mathcal{A}_n|,
\]
be the ordinary generating function of a combinatorial class $\mathcal{A}$, and let $\rho$ be its radius of convergence.

\begin{definition}
For a parameter $x \in (0,\rho)$, the \emph{Boltzmann distribution} on $\mathcal{A}$ is
\[
  \Pr_x(\gamma) = \frac{x^{|\gamma|}}{A(x)}.
\]
A random generator with this law is called a \emph{Boltzmann sampler}.
\end{definition}

The normalization is immediate:
\[
  \sum_{\gamma \in \mathcal{A}} \Pr_x(\gamma)
  = \frac{1}{A(x)} \sum_{n \ge 0} a_n x^n = 1.
\]
Every object of the same size receives the same probability. Therefore, conditioning on the event $|\gamma|=n$ produces a uniform random element of $\mathcal{A}_n$.

The size itself is random, with distribution
\[
  \Pr_x(|\gamma|=n)=\frac{a_n x^n}{A(x)}.
\]
Its mean is
\[
  \mathbb E_x[N] = \frac{xA'(x)}{A(x)}.
\]
The mean is monotone in $x$ because
\[
  \frac{d}{dx}\left(\frac{xA'(x)}{A(x)}\right) = \frac{\operatorname{Var}_x(N)}{x} \ge 0.
\]
So increasing $x$ pushes mass toward larger objects.

\section{Recursive Construction from Symbolic Specifications}

The symbolic method translates directly into Boltzmann samplers.

\paragraph{Disjoint union.}
If
\[
  \mathcal{A} = \mathcal{B} + \mathcal{C},
  \qquad A(x)=B(x)+C(x),
\]
then choose the $\mathcal B$ branch with probability $B(x)/A(x)$ and the $\mathcal C$ branch with probability $C(x)/A(x)$.

\paragraph{Cartesian product.}
If
\[
  \mathcal{A} = \mathcal{B} \times \mathcal{C},
  \qquad A(x)=B(x)C(x),
\]
then sample independently from the Boltzmann laws on $\mathcal B$ and $\mathcal C$ and return the pair.

\paragraph{Sequence.}
If
\[
  \mathcal{A} = \SEQ(\mathcal{B}),
  \qquad A(x)=\frac{1}{1-B(x)},
\]
with $B(x)<1$, then the number of components has geometric law
\[
  \Pr(K=k) = (1-B(x))B(x)^k,
  \qquad k \ge 0.
\]
After drawing $K$, sample $K$ independent Boltzmann objects from $\mathcal B$ and concatenate them.

These are the constructors we need most often. More delicate labeled and cycle constructions exist as well, but they are better treated in a specialized reference than in a one-page summary. The important point here is that the symbolic specification and the sampling rule use the same algebra.

\section{Tuning and Exact-Size Rejection}

If the target is exact size $n$, a natural choice of parameter $x$ is the one that makes the expected size equal to $n$:
\[
  \frac{xA'(x)}{A(x)} = n.
\]
This same equation also appears by maximizing the exact-size probability
\[
  \Pr_x(N=n)=\frac{a_n x^n}{A(x)}
\]
with respect to $x$: differentiate
\[
  n\log x - \log A(x)
\]
and set the derivative to zero.

This tuning equation need not have a solution for every class and every large $n$; it depends on the expected size actually diverging as $x \uparrow \rho$. After tuning $x$, one may simply rerun the Boltzmann sampler until the drawn object has the desired size. In many tree-like and admissible classes this is extremely efficient, but the precise complexity depends on local limit behavior and class-specific regularity assumptions. We deliberately avoid stating a one-line universal theorem here.

\section{The Gibbs Interpretation}

Now make the substitution
\[
  x=e^{-\beta},
\]
with $\beta$ interpreted as inverse temperature. Then
\[
  \Pr_\beta(\gamma) = \frac{e^{-\beta |\gamma|}}{A(e^{-\beta})}.
\]
If we define the energy of a structure to be its size,
\[
  E(\gamma)=|\gamma|,
\]
then this becomes
\[
  \Pr_\beta(\gamma)=\frac{e^{-\beta E(\gamma)}}{Z(\beta)},
  \qquad
  Z(\beta)=A(e^{-\beta}).
\]

So in this exact combinatorial setting:
\begin{itemize}
  \item the generating function is the partition function,
  \item Boltzmann weights are Gibbs weights,
  \item and the size variable is the energy.
\end{itemize}

This exactness depends on the particular choice of energy $E(\gamma)=|\gamma|$. If one changes the energy, one changes the partition function accordingly.

\section{Critical Temperature}

The radius of convergence determines the critical inverse temperature:
\[
  \beta_c = -\log \rho.
\]
For
\[
  \beta > \beta_c
  \qquad (x=e^{-\beta}<\rho),
\]
the partition function is finite and the Gibbs distribution is well defined. At the boundary
\[
  \beta = \beta_c
  \qquad (x=\rho),
\]
convergence depends on the coefficient tail.

If
\[
  a_n \sim C\rho^{-n} n^{-\alpha},
\]
then at criticality
\[
  a_n \rho^n \sim C n^{-\alpha}.
\]
So:
\begin{itemize}
  \item if $\alpha>1$, the critical partition function still converges;
  \item if $\alpha \le 1$, it diverges.
\end{itemize}

Even when the critical partition function converges, the expected size may still diverge. For example, if $\alpha=3/2$, then
\[
  \sum_n a_n \rho^n \sim \sum_n n^{-3/2}
\]
converges, but
\[
  \sum_n n a_n \rho^n \sim \sum_n n^{-1/2}
\]
diverges. This is why critical Boltzmann samplers often produce heavy-tailed sizes.

The physically relevant ``high-temperature'' limit inside the normalizable regime is therefore not ``$x$ close to $1$'' in general. It is the approach to criticality
\[
  x \uparrow \rho
  \qquad\text{or equivalently}\qquad
  \beta \downarrow \beta_c.
\]

\section{A Free-Energy Viewpoint}

It is sometimes useful to isolate the size-$n$ contribution
\[
  Z_n(\beta) = a_n e^{-\beta n}.
\]
This is the size-restricted partition weight. Its exponential rate is governed by
\[
  \frac{1}{n}\log Z_n(\beta) = -\beta + \frac{1}{n}\log a_n.
\]
If
\[
  a_n \sim C\rho^{-n}n^{-\alpha},
\]
then
\[
  \lim_{n\to\infty} \frac{1}{n}\log a_n = \log(1/\rho),
\]
so the leading exponential growth/decay rate is
\[
  -\beta + \log(1/\rho).
\]
This is a combinatorial growth exponent. One may call the negative of this quantity a free energy per unit size if desired, but the important point for this chapter is simpler: the same radius of convergence that controls coefficients also controls the thermodynamic threshold.

We deliberately avoid calling this quantity an entropy rate here. Chapter~9 showed that entropy-rate identities require a carefully specified probabilistic source and a carefully specified counting object.

\section{Worked Examples}

\begin{example}[Binary strings]
For binary strings,
\[
  A(z)=\frac{1}{1-2z},
  \qquad \rho=\frac{1}{2}.
\]
The Boltzmann sampler draws a geometric length $N$ with
\[
  \Pr_x(N=n) = (1-2x)(2x)^n,
\]
and then chooses a uniformly random binary string of that length. The expected size is
\[
  \mathbb E_x[N] = \frac{2x}{1-2x}.
\]
The partition function is
\[
  Z(\beta)=A(e^{-\beta})=\frac{1}{1-2e^{-\beta}},
\]
so
\[
  \beta_c = \log 2.
\]
\end{example}

\begin{example}[Catalan structures]
For Catalan objects,
\[
  C(z)=\frac{1-\sqrt{1-4z}}{2z},
  \qquad \rho=\frac{1}{4},
\]
and
\[
  [z^n]C(z) \sim \frac{4^n}{\sqrt{\pi}n^{3/2}}.
\]
At criticality,
\[
  \Pr_{\beta_c}(N=n) \propto [z^n]C(z)\rho^n \sim \frac{1}{\sqrt{\pi}n^{3/2}}.
\]
So the partition function is finite, but the expected size diverges. This is the same $n^{-3/2}$ exponent that earlier chapters encountered through square-root singularities.
\end{example}

\section*{Preview of Chapter 14}

There is a tempting analogy between Boltzmann sampling and temperature in autoregressive language models. For a global string distribution $\mu(w)$, the Gibbs-style reweighting
\[
  \mu_\beta(w) \propto \mu(w)^\beta
\]
really is a partition-function construction. Practical LLM temperature decoding, however, is usually applied \emph{locally} at each next-token distribution. Those two operations are related, but they are not identical in general. Chapter~14 will focus on that distinction rather than blur it.

\section{Exercises}

\begin{exercise}
For a combinatorial class with generating function $A(x)$, prove the variance identity
\[
  \operatorname{Var}_x(N) = x\frac{d}{dx}\left(\frac{xA'(x)}{A(x)}\right).
\]
Deduce that the expected size is monotone nondecreasing in $x$.
\end{exercise}

\begin{exercise}
For binary strings, solve
\[
  \frac{2x}{1-2x}=n
\]
for the tuned parameter $x=x^*(n)$.
\end{exercise}

\begin{exercise}
Show that for Catalan structures at criticality,
\[
  \sum_{n \ge 0} [z^n]C(z)\rho^n < \infty
\]
but
\[
  \sum_{n \ge 0} n [z^n]C(z)\rho^n = \infty.
\]
Interpret this probabilistically.
\end{exercise}
