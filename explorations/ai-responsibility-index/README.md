# AI Responsibility Index

An index for assessing corporate AI responsibility, drawing on ESG frameworks and LLM-based extraction methods.

## Goal

Build a system that can:
1. Define a structured set of AI responsibility indicators (drawing from CSIRO/Alphinity RAI-ESG framework)
2. Automatically extract evidence from corporate disclosures using LLMs (drawing from ESGReveal)
3. Score companies on their responsible AI practices

## Key References

- CSIRO/Alphinity RAI-ESG Framework (2024) - methodology and indicator design
- "Integrating ESG and AI" (Springer, 2025) - peer-reviewed version with full scoring methodology
- ESGReveal (Zou et al., 2024) - LLM+RAG implementation for structured ESG data extraction
