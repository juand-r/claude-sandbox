# Dynamical Systems, Ergodic Theory, and Reservoir Computing: An Introduction

An introductory expository textbook for mathematics undergraduates. The narrative arc: dynamical systems provide the geometric and analytic tools for understanding evolving systems, ergodic theory reveals their long-term statistical behavior, and reservoir computing harnesses dynamical systems as computational substrates — where stability, mixing, and entropy directly determine computational power.

## Prerequisites

- Linear algebra
- Multivariable calculus
- Basic real analysis (limits, continuity, sequences/series)
- Some exposure to ODEs is helpful but not required

## Format

Markdown with LaTeX math (inline `$...$`, display `$$...$$`). Each chapter includes exposition, definitions, theorems with proofs, worked examples, exercises, references, and recommended reading.
